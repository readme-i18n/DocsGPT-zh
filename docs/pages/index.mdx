#  **DocsGPT 🦖**

DocsGPT 是一款开源生成式AI工具，可帮助用户从任何知识源获取可靠答案，同时避免幻觉生成。它具备快速可靠的信息检索能力，并内置工具链与智能体系统功能。

### 核心特性:

- **🗂️ 多格式支持:** 可读取 PDF、DOCX、CSV、XLSX、EPUB、MD、RST、HTML、MDX、JSON、PPTX 及图片文件
- **🌐 网络数据集成:** 支持从URL、站点地图、Reddit、GitHub及网络爬虫获取数据
- **✅ 可靠答案:** 通过清晰UI界面获取附带来源引用的准确无幻觉回答
- **🔑 简化API密钥:** 生成与您的设置、文档和模型关联的密钥，简化聊天机器人及集成配置
- **🔗 可执行工具链:** 连接API、工具及其他服务以实现LLM操作
- **🧩 预置集成组件:** 提供开箱即用的HTML/React聊天组件、搜索工具、Discord/Telegram机器人等
- **🔌 灵活部署:** 兼容主流LLM（OpenAI、Google、Anthropic）及本地模型（Ollama、llama_cpp）
- **🏢 安全可扩展:** 支持Kubernetes私有化安全部署，满足企业级可靠性要求

**贡献与扩展：** 作为开源项目，我们非常鼓励社区贡献！如果您开发了有价值的自定义功能或增强特性，请考虑将其贡献回主仓库，让其他 DocsGPT 用户受益。
