# 为应用配置本地语言模型

为应用配置本地语言模型可显著增强其功能，使其无需依赖外部API即可理解和生成多语言文本。通过集成本地语言模型，您可以提升隐私性、降低延迟，并确保在离线环境下仍能持续运行。以下是关于如何为应用程序设置本地语言模型的完整指南：

## 操作步骤：

### 云端版本LLM更换：

<Steps >
### 第一步
访问聊天界面，您将看到默认选中的LLM模型。
### 第二步
点击该模型，将出现可供选择的各种LLM下拉列表。
### 第三步
选择您需要的LLM模型。

</Steps>

### 开源版本LLM更换：

<Steps>
### 第一步
对于开源版本，请编辑.env文件中的`LLM_PROVIDER`、`LLM_NAME`等参数。更多信息请参考[⚙️ 应用配置](/Deploying/DocsGPT-Settings)。
### 第二步
访问[☁️ 云服务提供商](/Models/cloud-providers)获取最新的在线模型列表。请确保您拥有正确的API_KEY和LLM_PROVIDER。
如需自托管方案，请访问[🖥️ 本地推理](/Models/local-inference)。
</Steps>
