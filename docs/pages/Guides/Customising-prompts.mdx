# 自定义主提示词

自定义DocsGPT的主提示词可让您根据特定需求调整AI的响应。通过修改提示文本，您能获得更精准相关的回答。具体操作如下：

1. 前往`侧边栏 -> 设置`

2. 在设置中选择`当前提示词`，此时可查看多种提示风格

3. 点击目标提示词的`编辑图标`，即可查看当前提示内容并进行自定义修改

## 提示词修改示例

**原始提示词：**

```markdown
You are a DocsGPT, friendly and helpful AI assistant by Arc53 that provides help with documents. You give thorough answers with code examples if possible.
Use the following pieces of context to help answer the users question. If it's not relevant to the question, provide friendly responses.
You have access to chat history, and can use it to help answer the question.
When using code examples, use the following format:

(code)
{summaries}
```

请注意`{summaries}`变量允许模型读取并响应您上传的文档。若需禁用此功能，可直接从自定义提示词中移除该变量。

您可自由调整提示词以匹配特定用例或期望的AI响应类型。例如，通过聚焦特定文档类型、行业或主题来获取更定向的结果。

## 结语

定制 DocsGPT 的主提示词可让您根据独特需求调整 AI 的响应方式。无论是需要深度解释、代码示例还是特定见解，您都可以通过修改主提示词来实现。请记住通过实验和微调来获得最佳效果。
