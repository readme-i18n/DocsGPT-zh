# 快速入门

**先决条件:**

* **Docker:** 确保您的系统已安装并运行 Docker。

## 启动 DocsGPT (macOS 和 Linux)

启动 DocsGPT 最简单的方式是使用提供的 `setup.sh` 脚本。该脚本可自动完成配置流程并提供多种安装选项。

**操作步骤:**

1. **下载 DocsGPT 仓库：**

   首先，您需要将 DocsGPT 仓库下载到本地计算机。可以使用 Git 命令完成：

   ```bash
   git clone https://github.com/arc53/DocsGPT.git
   cd DocsGPT
   ```

2. **运行 `setup.sh` 脚本：**

   在终端中进入 DocsGPT 目录，执行 `setup.sh` 脚本：

   ```bash
   ./setup.sh
   ```

3. **跟随交互式设置向导：**

   `setup.sh` 脚本会引导您完成交互式菜单，提供以下选项：

   ```
   Welcome to DocsGPT Setup!
   How would you like to proceed?
   1) Use DocsGPT Public API Endpoint (simple and free)
   2) Serve Local (with Ollama)
   3) Connect Local Inference Engine
   4) Connect Cloud API Provider
   Choose option (1-4):
   ```

   各选项说明如下：

   * **1) 使用 DocsGPT 公共 API 端点（简单免费）：** 这是最快捷的入门方式，无需 API 密钥或本地模型下载，直接调用 DocsGPT 公共 API。

   * **2) 本地运行（通过 Ollama）：** 此选项允许使用 [Ollama](https://ollama.com/) 在本地运行大语言模型。系统会提示选择 CPU/GPU 运行模式并下载模型，适合本地处理和实验。

   * **3) 连接本地推理引擎：** 如果您已部署 Llama.cpp、Text Generation Inference (TGI)、vLLM 等本地推理引擎，选择此选项后需指定引擎类型并提供连接参数。

   * **4) 连接云端 API 提供商：** 可对接 OpenAI、Google (Vertex AI/Gemini)、Anthropic (Claude)、Groq、HuggingFace Inference API 或 Azure OpenAI 等商业云 API，需提供对应平台的 API 密钥。

   完成选项配置（如提供 API 密钥或模型名称）后，脚本会自动生成 `.env` 文件并通过 Docker Compose 启动 DocsGPT。

4. **在浏览器中访问 DocsGPT：**

   当 Docker 容器运行后，在浏览器中访问 [http://localhost:5173/](http://localhost:5173/) 即可使用 DocsGPT 网页应用。

5. **停止 DocsGPT 服务：**

   在 `DocsGPT` 目录下执行以下命令即可停止服务：

   ```bash
   docker compose -f deployment/docker-compose.yaml down
   ```
   （或根据 `setup.sh` 执行结束时显示的特定 `docker compose` 命令，该命令可能包含您所选配置对应的附加 compose 文件）。

## 启动 DocsGPT (Windows 版)

针对 Windows 用户，我们提供了一个 PowerShell 脚本，其功能与 macOS/Linux 设置脚本相同。

**操作步骤：**

1. **下载 DocsGPT 仓库：**

   首先需要将 DocsGPT 仓库下载到本地计算机，可通过 Git 执行：

   ```powershell
   git clone https://github.com/arc53/DocsGPT.git
   cd DocsGPT
   ```

2. **运行 `setup.ps1` 脚本：**

   执行 PowerShell 设置脚本：

   ```powershell
   PowerShell -ExecutionPolicy Bypass -File .\setup.ps1
   ```

3. **跟随交互式设置向导：**

   与 Linux/macOS 脚本类似，PowerShell 脚本会引导您完成 DocsGPT 的配置。
   该脚本将处理环境配置并根据您的选择启动 DocsGPT。

4. **在浏览器中访问 DocsGPT：**

   当设置完成且 Docker 容器运行后，在浏览器中访问 [http://localhost:5173/](http://localhost:5173/) 即可使用 DocsGPT 网页应用。

5. **停止 DocsGPT：**

   要停止 DocsGPT，请运行设置脚本执行结束时显示的 Docker Compose down 命令。

**Windows 重要提示：** 请确保 Docker Desktop 已正确安装并在 Windows 系统上运行。脚本会尝试在 Docker 未运行时启动它，但若遇到问题可能需要手动启动。

**替代方法：**
如果您倾向于更手动的方式，可以按照我们的 [Docker 部署文档](/Deploying/Docker-Deploying) 获取详细指南，了解如何直接在 Windows 上使用 Docker 命令设置 DocsGPT。
