# 集成 DocsGPT 搜索栏小部件

## 项目说明

DocsGPT 搜索栏小部件提供了一种简单而强大的方式，将 AI 驱动的文档搜索直接嵌入到您的 Web 应用程序中。该小部件允许用户跨文档或页面执行搜索，帮助他们快速找到所需信息。本指南将引导您完成将搜索栏小部件嵌入项目的全过程，无论您使用的是 React 还是纯 HTML。

在 [DocsGPT Widget Demo](https://widget.docsgpt.cloud/) 上试用交互式小部件展示并自定义其参数。

## React Setup

### Installation

Make sure you have Node.js and npm (or yarn, pnpm) installed in your project.  Navigate to your project directory in the terminal and install the `docsgpt` package:

```bash npm
npm install docsgpt
```

### Usage

In your React component file, import the `SearchBar` component:

```js
import { SearchBar } from "docsgpt";
```

Now, you can embed the widget within your React component's JSX:

```jsx
<SearchBar
    apiKey="your-api-key"
    apiHost="https://your-docsgpt-api.com"
    theme="light"
    placeholder="Search or Ask AI..."
    width="300px"
/>
```

### Installation

To use the DocsGPT Search Bar Widget directly in HTML, include the widget script from a CDN in your HTML file:

```html filename="html"
<script
  src="https://unpkg.com/docsgpt/dist/legacy/main.js"
  type="module"
></script>
```

### Usage

In your HTML `<body>`, add a `<div>` element where you want to render the Search Bar Widget. Set an `id` for easy targeting.

```html filename="html"
<div id="search-bar-container"></div>
```

Then, in a `<script type="module">` block, use the `renderSearchBar` function to initialize the widget, passing the `id` of your `<div>` and a configuration object. To link the widget to your DocsGPT API and configure its behaviour, pass the relevant parameters within the configuration object of `renderSearchBar`.

```html filename="html"
<!DOCTYPE html>
<div id="search-bar-container"></div>
<script type="module">
  window.onload = function() {
    renderSearchBar('search-bar-container', {
      apiKey: 'your-api-key-here',
      apiHost: 'https://your-api-host.com',
      theme: 'light',
      placeholder: 'Search here...',
      width: '300px'
    });
  }
</script>
```

## 属性表格

DocsGPT 搜索栏小部件提供了一系列可自定义属性，使您能够调整其外观和行为以完美匹配您的 Web 应用程序。这些参数可以在将小部件嵌入 React 组件或 HTML 代码时直接修改。以下是每个可用属性的详细概述：

| **属性**       | **类型**  | **默认值**                   | **描述**                                                                                  |
|-----------------|-----------|-------------------------------------|--------------------------------------------------------------------------------------------------|
| **`apiKey`**    | `string`  | `"your-api-key"`                  | 用于 DocsGPT API 身份验证的 API 密钥。如果不需要身份验证，请留空。                         |
| **`apiHost`**   | `string`  | `"https://gptcloud.arc53.com"`       | **必填。** 你的 DocsGPT API 后端 URL。该端点处理向量相似性搜索查询。           |
| **`theme`**     | `"dark" \| "light"` | `"dark"`                            | 搜索栏的颜色主题。可选值: `"dark"` 或 `"light"`。默认为 `"dark"`。                                     |
| **`placeholder`** | `string` | `"Search or Ask AI..."`             | 搜索输入框中显示的占位文本。                                           |
| **`width`**     | `string`  | `"256px"`                          | 搜索栏的宽度。接受任何有效的 CSS 宽度值 (例如 `"300px"`, `"100%"`, `"20rem"`)。 |

---

## 组件属性说明

*   **完全自定义:** 表格中列出的所有属性均可自定义。覆盖默认值以创建与您的品牌和应用场景完美匹配的 Search Bar Widget。
*   **API 密钥处理:** `apiKey` 属性为可选配置。仅当您的 DocsGPT 后端 API 需要密钥认证时才需填写。DocsGPT Cloud 的 `apiHost` 为 `https://gptcloud.arc53.com/`

## 深入探索与定制

DocsGPT Search Bar Widget 完全开源，支持超越现有属性的深度定制与扩展。

基于 React 的组件完整源代码位于主仓库 [DocsGPT GitHub Repository](https://github.com/arc53/DocsGPT) 的 `extensions/react-widget` 目录中。欢迎查阅代码、分叉仓库并按需定制组件。
